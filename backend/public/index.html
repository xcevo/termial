Q&A Section

Q: How to use the terminal?

A: Simply type your commands.

Q: What commands can I run?

A: Any shell commands supported by the backend.

Q: How do I list files in a directory?

A: Use ls to list files and directories.

Q: How do I navigate between directories?

A: Use cd directory_name to move into a directory and cd .. to go back.

Q: How do I check my current directory?

A: Use pwd to print the current working directory.

Q: How do I create a new directory?

A: Use mkdir directory_name to create a new folder.

Q: How do I create a new file?

A: Use touch filename or echo "content" > filename to create a file.

Q: How do I delete a file or directory?

A: Use rm filename to delete a file, and rm -r directory_name to remove a directory.

Q: How do I edit a file in the terminal?

A: You can use nano filename, vim filename, or vi filename to edit files.

Q: How do I view the contents of a file?

A: Use cat filename, less filename, or more filename.
